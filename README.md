## Rodando o Projeto Localmente

Para rodar este projeto na sua máquina, siga as instruções abaixo.

### Pré-requisitos

Antes de começar, certifique-se de que você tem o [Node.js](https://nodejs.org/en/) (versão 18 ou superior) instalado.

### Instalação

1.  **Clone o repositório:**

    ```bash
    git clone https://github.com/Chang3MyMind/projeto-teste
    ```

2.  **Vá até pasta do projeto:**

    ```bash
    cd projeto-teste
    ```

3.  **Instale as dependências:**

    ```bash
    npm install
    ```

4.  **Inicie o servidor:**

    ```bash
    npm run dev
    ```

5.  **Abra o projeto no navegador:**
    Acesse o [http://localhost:3000](http://localhost:3000) no seu navegador preferido para ver a aplicação.
